# Software

Seja bem-vindo à seção de software. Aqui você encontrará aplicativos de qualidade para Android, iOS, Windows e Linux. Alguns exemplos de como os softwares aqui listados podem ajudá-lo são: assistir anime ou ler mangá em qualquer lugar, tanto on-line (scrapping) quanto off-line (download), ler mangá em uma tela maior etc.
___

### Anime

- [**Dantotsu**](https://github.com/rebelonion/Dantotsu) [!badge variant="ghost" icon="video" text="TV"](https://github.com/siennaXD/SaikouTV/)  [!badge variant="ghost" icon="globe" text="Web"](https://dantotsu.app/)
- Sucessor espiritual do Saikou
- Interface de usuário decente, mas com personalização limitada
  - Pode ser usado off-line

- Interface de usuário minimalista e personalizável
  - Também oferece outros tipos de mídia
  - Suporte a rastreamento e backup
  - Suporte a jogadores locais
  - é necessário fazer parte do servidor Discord para obter os URLs de extensão
- [**Aniyomi**](https://github.com/jmir1/aniyomi/) [!badge variant="ghost" icon="globe" text="Web"](https://aniyomi.org/) [!badge variant="ghost" size="xs" text="Manga"]

  - Design limpo e minimalista
  - Também oferece outros tipos de mídia
  - Também possui leitor de mangá
  - Suporte de rastreamento com tipo aprimorado

### Manga

- [**Mihon**](https://mihon.app/download/) [!badge variant="ghost" icon="globe" text="Web"](https://mihon.app/) [!badge variant="ghost" icon="repo-forked" text="Forks"](https://tachiyomi.org/forks/)
  - Ótima interface de usuário personalizável
  - Toneladas de fontes de mangá
  - Suporte a download e leitor local
  - Suporte a rastreamento e sincronização
  - Alto consumo de cache
- [**Perfect Viewer**](https://play.google.com/store/apps/details?id=com.rookiestudio.perfectviewer&hl=en&gl=US) [!badge variant="ghost" size="xs" text="Local"]
  - O leitor em si é muito bom, com vários filtros
  - Tem suporte de plug-in para os serviços de nuvem, mas não para sites de mangá
  - Suporte a todos os formatos de quadrinhos.
  - O aplicativo é leve, mas a interface do usuário parece antiga
  - Alguns recursos estão protegidos por um acesso pago
- [**Kotatsu**](https://github.com/KotatsuApp/Kotatsu) [!badge variant="ghost" icon="globe" text="Web"](https://kotatsu.app/)
  - UI minimalista com personalização limitada
  - Oferece sugestões de mangás
  - Suporte a download e leitor local
  - Suporte a rastreamento e sincronização
  - Faltam alguns recursos básicos

### Light Novel

- [**Moon+ Reader Pro**](https://play.google.com/store/apps/details?id=com.flyersoft.moonreaderp&hl=en&gl=US)
  - UI personalizável
  - Suporte a fontes on-line
  - Analytics de leitura no aplicativo
  - Suporte a backup e restauração
  - Sem suporte a rastreadores
  - Muitos recursos são pagos, a versão gratuita exibe anúncios em banner
- [**ReadEra**](https://play.google.com/store/apps/details?id=org.readera&hl=en&gl=US)
  - Interface de usuário agradável com personalização limitada
  - A versão gratuita não tem anúncios.
  - Não há analytics de leitura no aplicativo
  - Boa parte dos recursos está protegida por um acesso pago
  - Sem suporte a fontes e rastreadores on-line
- [**Shosetsu**](https://gitlab.com/shosetsuorg/shosetsu) [!badge variant="ghost" icon="globe" text="Web"](https://shosetsu.app/)
  - UI minimalista com muita personalização
  - Tem suporte a CSS
  - Download lento e sem conversão de epub
  - Sem suporte a imagens, página única e rastreamento
- [**Quicknovel**](https://github.com/LagradOst/QuickNovel)
  - Boa interface de usuário com muita personalização, incluindo fontes
  - Rápido download e conversão de epub
  - Suporte a imagens (somente para fontes compatíveis)
  - Atualização lenta do aplicativo com fontes quebradas
  - Sem suporte a página única e rastreamento

## PC

___

### Manga

#### Online:

- [**Houdoku**](https://houdoku.netlify.app/) [!badge variant="ghost" size="xs" text="win"] [!badge variant="ghost" size="xs" text="mac"] [!badge variant="ghost" size="xs" text="lin"]
    - Fácil de configurar
    - Boa interface de usuário personalizável
    - Tem uma boa quantidade de extensões
    - Integração com o rastreador
    - Pode baixar mangás
- [**Tachidesk**](https://github.com/Suwayomi/Tachidesk-Server) [!badge variant="ghost" size="xs" text="win"] [!badge variant="ghost" size="xs" text="mac"] [!badge variant="ghost" size="xs" text="lin"]
    - Usa as mesmas extensões e backups que o tachiyomi, portanto, oferece bom suporte a extensões
    - A interface do usuário é igual à do tachiyomi
    - Pode baixar mangás
    - Não há integração com rastreadores

=== Clients
- [**Tachidesk base**](https://github.com/Suwayomi/Tachidesk-Server)
    - Muitas fontes para escolher
    - configurações básicas para leitura
    - carece de recursos básicos de leitura
    - requisitos complicados, como JAVA
- [**Sorayomi**](https://github.com/Suwayomi/Tachidesk-Sorayomi)
    - Necessita do servidor Tachidesk para funcionar
    - Mínimo e fácil de usar, desatualizado em 2022
    - Recursos insuficientes
    - carrega lentamente (depende de sua internet)
- [**Tachidesk JUI**](https://github.com/Suwayomi/Tachidesk-JUI)
    - Precisa do servidor Tachidesk para funcionar
    - Download de mangás em lote

===

#### Offline:

- [**CDisplayEx**](https://www.cdisplayex.com/) [!badge variant="ghost" size="xs" text="win"]
    - UI minimalista
    - Leitor rápido e simples com boa quantidade de configurações
    - Tem mais opções de redimensionamento
    - Não há modo de leitura vertical
- [**YACReader**](https://www.yacreader.com/) [!badge variant="ghost" size="xs" text="win"] [!badge variant="ghost" size="xs" text="mac"] [!badge variant="ghost" size="xs" text="lin"]
    - Rápido e personalizável
    - Vem com o YAClibrary, que é como o Calibre para quadrinhos
    - Sem modo de leitura vertical
- [**OpenComic**](https://github.com/ollm/OpenComic) [!badge variant="ghost" size="xs" text="win"] [!badge variant="ghost" size="xs" text="mac"] [!badge variant="ghost" size="xs" text="lin"]
    - UI decente
    - O tempo de carregamento inicial é ligeiramente mais lento do que o dos dois leitores acima
    - Tem modo de leitura vertical com quantidade limitada de personalização

## iOS

___

### Anime

- [**Anime Now**](https://github.com/AnimeNow-Team/AnimeNow)
  - Ótima interface de usuário com suporte a extensões
  - Suporte ao download de episódios
  - Suporte a video casting
  - Suporte a rastreamento
  - Disponível somente por meio de sideload
- [**9Animator**](https://github.com/SuperMarcus/NineAnimator)
  - Interface de usuário decente com integração de extensões
  - Suporte a transmissão e notificação
  - Suporte a rastreamento
  - Episódio de download disponível
  - Pequeno número de extensões
  - Disponível somente por meio de sideload

### Manga

- [**Paperback**](https://github.com/Paperback-iOS/app) [!badge variant="ghost" icon="globe" text="Web"](https://paperback.moe/)
  - UI muito limpa com tema mutável
  - Suporte a rastreador e sincronização
  - Muitas opções diferentes de download
  - Lançamento na AppStore somente para alguns países
- [**Aidoku**](https://github.com/Aidoku/Aidoku) [!badge variant="ghost" icon="globe" text="Web"](https://aidoku.app/)
  - Disponível no Testflight (não é necessário fazer sideload)
  - Muitas fontes
  - Suporte a rastreadores
  - Possibilidade de download de capítulos
  - Ainda em desenvolvimento
- [**Hanami**](https://github.com/oolxg/Hanami)
  - Disponível no Testflight (não é necessário fazer sideload)
  - Interface de usuário razoável com alguns atrasos
  - Apenas o MangaDex como fonte. Portanto, não há lançamentos oficiais
- [**YACReader**](https://apps.apple.com/app/id635717885) [!badge variant="ghost" size="xs" text="Local"] [!badge variant="ghost" size="xs" text="Pago"]
  - Várias opções de transferência e suporte a leitores locais
  - Suporte aos formatos cbz, cbr, zip, tar, arj, pdf e rar, além de formatos de imagem
  - Capacidade de fazer backup de histórias em quadrinhos
- [**SideBooks**](https://apps.apple.com/us/app/sidebooks/id409777225) [!badge variant="ghost" size="xs" text="Cloud"]
  - Compatível com PDF, ZIP, CBZ, RAR e CBR
  - Possui todos os recursos necessários
  - UI de aparência antiga, mas muito funcional e personalizável.
  - Boa quantidade de recursos com paywall
